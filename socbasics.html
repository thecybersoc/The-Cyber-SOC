SOC Fundamentals

Beginner to SOC-ready: Learn tools, processes, and incident handling

Course Overview

The SOC Fundamentals course is designed for beginners to understand the world of Security Operations Centers. This course covers cybersecurity foundations, network essentials, SOC tooling, incident response, and reporting. Students will gain hands-on experience with industry-standard tools and processes to become confident SOC analysts.

Course Roadmap

Week 1 – SOC & Cybersecurity Foundations
Class 1: Introduction to SOC and Cybersecurity Concepts
- What is a SOC
- SOC roles and responsibilities
- Threats, vulnerabilities, and risks
- Overview of cyberattack lifecycle

Class 2: Threat Landscape and Security Governance
- Cybercriminals and categories of attacks
- Types of malware and network attacks
- Business and security policies
- Introduction to security frameworks and compliance

Week 2 – Networking Essentials for SOC Analysts
Class 3: Networking Fundamentals
- OSI and TCP/IP models
- Data flow and encapsulation
- IP addressing and ICMP
- ARP overview

Class 4: Core Network Components
- TCP vs. UDP overview
- Routers, switches, and firewalls
- Introduction to IDS/IPS devices
- Understanding how traffic flows through the network

Week 3 – Security Controls and SOC Tooling
Class 5: Network and Endpoint Security
- Firewalls and intrusion detection/prevention systems
- Endpoint security basics
- Authentication, Authorization, and Accounting
- Introduction to Zero Trust Security

Class 6: SIEM Fundamentals
- What is a SIEM and how it supports a SOC
- Common log sources
- Log structure and interpretation basics
- Understanding alert generation and triage

Week 4 – Incident Response and Reporting
Class 7: Incident Handling Process
- Introduction to incident response lifecycle
- Evidence handling and documentation basics
- Overview of vulnerability scoring

Class 8: Reporting and Communication
- Writing a simple SOC incident report
- Key elements of escalation and communication within a SOC
- Introduction to digital forensics
- Course wrap-up
